# Choose Me — Horror Glitch Page

Одностраничный хоррор-сайт с глитч-эффектами, дрожанием букв и киберпанк-атмосферой.

## Файлы

- `index.html` — единственный файл, всё внутри (HTML + CSS + JS)

## Как запустить локально

```bash
# Просто открой файл в браузере
open index.html
```

## Деплой на GitHub Pages

1. Создай новый репозиторий на GitHub
2. Загрузи `index.html` и `README.md`
3. Перейди в **Settings → Pages**
4. В разделе **Source** выбери ветку `main`, папку `/ (root)`
5. Нажми **Save**
6. Сайт будет доступен по адресу: `https://ТВО_ЛОГИН.github.io/ИМЯ_РЕПО/`

## Эффекты

- Каждая буква дрожит независимо (spring-физика)
- Рандомные буквы глитчат: быстро сменяют символы/цифры с вспышкой цвета
- Хроматическая аберрация (циановый/маджентовый сдвиг)
- Полосы помех поперёк экрана
- Мерцание всей надписи
- Кастомный курсор с дрожанием
- Сканлайны поверх экрана
- Шрифт: Special Elite (Google Fonts, загружается автоматически)

## Зависимости

Только Google Fonts (Special Elite) — подгружается онлайн.  
Всё остальное работает без интернета.
